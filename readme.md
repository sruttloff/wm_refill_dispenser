Wavemaker - Automatische RO Wasser Nachfüllung - 1-Kanal Dosierpumpe
=======
Basierend auf dem Wavemaker Projekt soll nun noch der Wasserstand des Nachfülltanks überwacht werden.
Ist hier ein niedriges Limit erreicht soll mit einer Schlauchpumpe Wasser aus einem Reservoir nachgefüllt werden.
Des Weiteren soll eine Dosierpumpe Tropfenweise Plankton dosieren.
